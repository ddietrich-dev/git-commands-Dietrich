# my little git-commands-projekt
these are my learning lessons for the most important commands in Git and GitHub
## git init
initiates a new, local git-repository in the current directory
```bash
git init
```

## git clone
gets data from a server - for example gitHub - on my own (virtual) server
```bash
git clone
```

## git add
marks changed files to prepare them for the next commit
```bash
git add
```

## git commit
creates a snapshot of each changing process in the projekt history
```bash
git commit
```

## git push
uploads my local commits on a online server - for example on GitHub
```bash
git push
```

## git pull
downloads latest changes from the server to my local projekt
```bash
git pull
```

## git remote
manages the connection between my local folder an the online server
```bash
git remote
```
